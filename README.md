# darksynthwave

A GPL-licensed daisyUI theme package for the Burnsedia visual style.

- Brand names and logos are covered by `TRADEMARKS.md`

## Usage

Install dependencies:

```bash
npm install
```

Build CSS:

```bash
npm run build
```

Watch mode:

```bash
npm run dev
```
